# CHANGELOG

## [1.2.0](https://github.com/team-telnyx/ios-telnyx-voice-ai-widget/releases/tag/1.2.0) (2025-12-02)

### Features
- **UI Redesign**: Updated widget interface with improved message bubbles, input bar styling, and avatar images
- **Overflow Menu**: Added menu with additional actions in conversation view

### Bug Fixes
- **iOS 13 Compatibility**: Fixed compatibility issues with iOS 13 devices
- **Image Picker**: Fixed camera and photo library sheet presentation issues
- **Transcript Display**: Resolved issue where empty AI messages appeared in conversation

## [1.1.0](https://github.com/team-telnyx/ios-telnyx-voice-ai-widget/releases/tag/1.1.0) (2025-11-17)

### Features
- **Image Support**: Widget now supports sending images during conversations
- **WebRTC SDK Update**: Updated to TelnyxRTC SDK version 2.4.0


## [1.0.0](https://github.com/team-telnyx/ios-telnyx-voice-ai-widget/releases/tag/1.0.0) (2025-10-30)

### Features
- **Initial Release**: Drop-in iOS widget for integrating Telnyx AI Assistant functionality with SwiftUI, featuring real-time audio visualization, multiple UI states, voice controls, and full conversation transcript support
